Intro to HTML Formatting

During the course of the classes taken on HTML, I learnt about tags,elements and their attributes. I learnt that some html tags are opening and self-closing(e.g the img tag) about opening and closing tags of the html elements.
The html elements are used to design the content placed in it and that the html language is the skeletal framework of the web.

Tags and its function

  • Header tags - This tags contains all information about the header which includes the description of the webpage, the type of document and its encoding.
  • Body tag - This tags contains all information about the body which includes all the information and content which would be passed into that document.
  • H1 - H6 tag - The size of a font decreases in progression to the increase of its number.
  • a href - This tag carries within the link of a web address. A Link
  • b - The b tag is to make a text content bold.
  • i - The i tag is to make a text content italic.
  • u - The u tag is to make a text content underline.
  • em - The em tag is to give a text content more emphasis than normal text.
  • img - The img tag carries within an image document to be displayed on the.
  • br - The
    br tag is give a line break in a document.
  • mark - This tag marks out a text content on webpage.
  • del - This tag is used to signify a text deleted on the webpage with less relevance.
  • ins -The ins tag is used to insert text into the page.
  • sup - This tag is used to display a number or value with square (e.g 102).
  • sub - This tag is used to display a number or value with a root value (e.g H2O).
  • input - This tag is used to collect information from a webpage (it usually contains within it text, number, email, radio-button.etc).
  • button - The button tag is used to assign an action to a particular event.
